Car Price Prediction using Machine Learning

This project is a Car Price Prediction System built using Python and Machine Learning.
The model predicts the resale price of a used car based on features like company, model, year, fuel type, and kilometers driven.

📌 Features
Predicts used car prices accurately
Data preprocessing using Pipelines
Machine Learning model training & testing
User input-based price prediction
Clean and structured dataset handling

🛠️ Technologies Used
Python
Pandas
NumPy
Scikit-learn
Jupyter Notebook

📂 Dataset Source

The dataset used in this project was collected from the Quickr Cars platform and further cleaned & preprocessed for Machine Learning model training.

📊 Model Performance
Best R² Score: 89.99%
Machine Learning algorithm used for accurate prediction and evaluation.
